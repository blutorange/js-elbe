  • Inherited from Stream.chunk

    stream([1,2,3,4,5,6,1,2]).chunk(i => Math.floor((i-1) / 3)) // => Stream[ [1,2,3], [4,5,6], [1,2] ]

    Type parameters

    • K

      Type of the returned value of the chunker,

    Parameters

    • classifier: BiFunction<Try<T>, number, K>

      It is passed the item as its first argument and the index as its second. Items are chunked together according to the returned value.

    Returns Stream<Try<T>[]>

    A stream over the chunked items.

    Parameters

    • finisher: Function<S, R>
